# SEQUENCE CLASSIFIER

Ссылка на задачу: https://www.codewars.com/kata/5921c0bc6b8f072e840000c0

При написании тестов использовался фреймворк Jest.

## Инициализация

```
npm install
```

## Запуск UNIT и интеграционных тестов

```
cd ./frontend/
npm test
```

## Запуск e2e тестов
```
cd ./frontend/
npx wdio run ./wdio.conf.js
```
